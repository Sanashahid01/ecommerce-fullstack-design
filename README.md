# VENDOORA - eCommerce Web Application

## Figma Design Template
- **Template:** Simple Ecommerce - Free Figma Website Template
- **Source:** Figma Community
- **URL:** https://www.figma.com/community/file/1358392116512811981/simple-ecommerce-free-figma-website-template
- **License:** 100% Free for personal and commercial use

## Design Inspiration
This project is based on the Figma template above with customizations:
- Brand name: "VENDOORA"
- Color scheme: Purple (#7B1FA2) + Orange (#FF7043)
- Products customized for Pakistani market (Rs. pricing)
- 24 products across 6 categories

## Pages Implemented
- Home Page
- Product Listing Page
- Product Details Page
- Cart Page

## Technologies Used
- Frontend: React.js, CSS3
- Backend: Node.js, Express.js
- Database: Firebase (planned for Week 2)

## Weekly Progress
- **Week 1:** Static frontend development with responsive design
- **Week 2:** Backend setup and dynamic integration (in progress)
- **Week 3:** Authentication, admin panel, and deployment (planned)
